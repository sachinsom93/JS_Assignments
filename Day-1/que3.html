Data types in js

1. string

A string (or a text string) is a series of characters.

eg. "sachin"

2. number

Numbers can be written with, or without decimals.

eg. let a = 3, let b = 3.3

3. Boolean

Booleans can only have two values: true or false.

let x = true, let y = false

4. Arrays

JavaScript arrays are written with square brackets.

Array items are separated by commas.

let fruits = ["mango", "apple", "banana"]

5. Objects

JavaScript objects are written with curly braces {}.

Object properties are written as name:value pairs, separated by commas.

let intro = { "name" : "sachin" , "city" : "jaipur" }

6. Undefined

In JavaScript, a variable without a value, has the value undefined.

let x; // here value of x is undefined

7. Null

In JavaScript null is "nothing". It is supposed to be something that doesn't exist.

var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"}; person = null; // Now value is null, but type is still an object
